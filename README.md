## 🧐 О проекте

**Aqbobek Portal** — это современная веб-платформа, объединяющая образовательный контент и передовые технологии искусственного интеллекта. Проект создан для обеспечения быстрого доступа к учебным материалам и оперативной помощи пользователям.

---

## 🤖 AI Ассистент

Одной из ключевых особенностей портала является **интегрированный умный помощник**:
* **Powered by Groq API:** Использование высокопроизводительных языковых моделей для мгновенных ответов.
* **Контекстная помощь:** Ассистент помогает пользователям ориентироваться на сайте, отвечает на вопросы об обучении и предоставляет справочную информацию.
* **Скорость:** Оптимизированная генерация текста в реальном времени.
